📡 M25 Monitor

PILOT Extension · v1.0

О расширении

M25 Monitor — это расширение для PILOT GPS, которое отображает только те объекты (транспортные средства), оборудованные трекерами модели M25. При выборе объекта в левой навигационной панели в основной области открывается внешняя страница мониторинга https://mega-info.su/dealer2/.

Функциональность

  • Автоматическая загрузка реальных данных из PILOT через /ax/tree.php?vehs=1&state=1.
  • Фильтрация объектов по полю equipment (содержит "M25", регистронезависимо).
  • Иерархическое дерево с сохранением групп, содержащих M25-объекты.
  • Колонки: Объект, IMEI, Оборудование.
  • При клике на объект в правой панели открывается iframe со страницей https://mega-info.su/dealer2/.
  • Панель инструментов: кнопка обновления iframe, кнопка открытия в новом окне (на случай блокировки iframe).
  • Обработка ошибок: если iframe не загружается из-за X-Frame-Options, показывается информационное сообщение и предлагается открыть в новом окне.

Установка и настройка

Расширение развёртывается на любом статическом хостинге (Cloudflare Pages, GitHub Pages, обычный веб-сервер). После размещения файлов убедитесь, что файл Module.js доступен по публичному URL.

  1. Загрузите все файлы расширения в корень вашего хостинга (например, https://your-domain.com/m25_monitor/).
  2. Проверьте доступность Module.js напрямую в браузере.
  3. В административной панели PILOT зарегистрируйте расширение:
    • Имя расширения: m25_monitor (только латиница, нижнее подчёркивание).
    • Базовый URL: https://your-domain.com/m25_monitor/.
  4. После регистрации PILOT будет проксировать все запросы через /store/m25_monitor/....

Примечание: Для работы фильтрации необходимо, чтобы в данных PILOT присутствовало поле equipment с указанием модели трекера.

Технические детали

Расширение написано в строгом соответствии с AI_SPECS.md. Код использует Ext JS 7.7+, компоненты Ext.tree.Panel, Ext.iframe (или обычный HTML iframe) и стандартные объекты PILOT: skeleton.navigation, skeleton.mapframe. Все асинхронные вызовы обрабатываются с проверкой ошибок. 

Ключевые файлы:
├── Module.js       — точка входа, инициализация навигации и главной панели
├── doc/
│   └── index.html  — эта документация
├── index.html      — статическая заглушка (для корректной работы Cloudflare)
└── style.css       — (опционально) дополнительные стили

Возможные проблемы и их решение

  • Нет объектов с M25 — убедитесь, что в вашем аккаунте PILOT есть транспорт с оборудованием, содержащим подстроку "M25".
  • Страница mega-info.su не открывается в iframe — сайт может запрещать встраивание (X-Frame-Options). Расширение показывает сообщение и предлагает открыть страницу в новой вкладке.
  • Не загружается дерево — проверьте, что расширение запущено внутри PILOT, а не как отдельная HTML-страница. Также убедитесь, что пользователь авторизован в системе.
