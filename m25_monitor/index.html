📡 M25 Monitor

Расширение для PILOT GPS — фильтрация и отображение объектов с трекерами модели M25.

📌 Это не самостоятельное приложение.
Расширение загружается и работает только внутри PILOT.
Если вы видите эту страницу, значит статические файлы расширения доступны, но для использования необходимо зарегистрировать расширение в PILOT.

📁 Файлы расширения

  • Module.js — точка входа (класс Store.m25_monitor.Module)
  • view/Navigation.js — левая панель с деревом M25-объектов
  • view/MainPanel.js — правая панель с iframe и тулбаром
  • doc/index.html — полная документация

🚀 Регистрация в PILOT

  1. Разместите все файлы на статическом хостинге (Cloudflare Pages, GitHub Pages и т.д.)
  2. Убедитесь, что /Module.js открывается в браузере
  3. В админке PILOT: Администрирование → Приложения → Добавить
  4. Имя (slug): m25_monitor (только латиница, нижнее подчёркивание)
  5. URL: https://ваш-хост/ (обязательно с завершающим слешем)
  6. Сохранить, включить для пользователя, перезагрузить PILOT (Ctrl+F5)

📖 Подробное описание, диагностика и примеры настроек — в документации.
