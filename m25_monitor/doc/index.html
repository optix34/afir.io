M25 Monitor Extension

Документация для администраторов и пользователей PILOT

Обзор

Расширение M25 Monitor добавляет в PILOT GPS левую вкладку «M25 Monitor». Вкладка отображает иерархическое дерево всех транспортных средств, у которых в поле equipment встречается подстрока M25 (регистр не важен). При выборе объекта справа загружается внешняя страница мониторинга https://mega-info.su/dealer2/.

Установка

1. Размещение файлов

Скопируйте следующую структуру файлов на ваш статический хостинг:

m25_monitor/
├── index.html
├── Module.js
├── doc/
│   └── index.html   (этот файл)
└── style.css        (опционально)

Убедитесь, что Module.js доступен по URL вида https://ваш-хост/m25_monitor/Module.js.

2. Регистрация в PILOT

Войдите в административную панель PILOT → раздел «Расширения» (Extensions). Добавьте новое расширение:

  • Имя расширения (slug): m25_monitor (только латиница, нижнее подчёркивание)
  • Базовый URL: https://ваш-хост/m25_monitor/ (обязательно с завершающим слэшем)

После сохранения PILOT будет автоматически проксировать /store/m25_monitor/... → ваш-хост/....

Использование

  • Перейдите в раздел M25 Monitor в левом меню навигации.
  • Разверните группы, чтобы увидеть объекты с оборудованием M25.
  • Кликните на любой объект — справа откроется iframe с внешней страницей.
  • Если страница не отображается (блокировка X-Frame-Options), нажмите кнопку «Открыть в новом окне» в правой панели.

Структура данных из PILOT

Расширение использует эндпоинт /ax/tree.php?vehs=1&state=1, который возвращает иерархический JSON. Пример узла транспортного средства: 

{
  "id": "veh_123",
  "text": "Грузовик 45",
  "type": "veh",
  "vehid": 12345,
  "imei": "865234012345678",
  "equipment": "M25-GLONASS",
  ...
}

Фильтрация ищет equipment.indexOf('M25') !== -1 (без учёта регистра).

Настройка API ключей (не требуется)

Данное расширение не использует внешние API, кроме встроенной страницы mega-info.su. Для корректной работы iframe достаточно сетевого доступа к этому домену.

Устранение неполадок

✓ Дерево пустое — проверьте, есть ли в системе объекты с equipment, содержащим «M25». Если нет — ни один объект не отобразится.
✓ Ошибка загрузки дерева — откройте консоль разработчика (F12) и посмотрите ответ от /ax/tree.php. Убедитесь, что вы авторизованы в PILOT.
⚠️ Страница не загружается в iframe — сайт mega-info.su может запрещать встраивание. Расширение покажет сообщение и предложит альтернативу. Это нормально.

Версия и совместимость

Расширение протестировано с PILOT версии 7.7+ (Ext JS 7.7). Требуется наличие глобальных объектов window.skeleton, skeleton.navigation, skeleton.mapframe.
